# AirSim (Real Time Data Monitor) StationGadgetGUI

## Abstract
AirSim is a project about realtime data monitoring of temperature and humidity data comming from multiples Raspberry Pis. This repository is a desktop UI for Windows.
This repository is a monitor client according to the following [server repository](https://github.com/kv-ohwtfyb/airsimHerokuServer). 

## User Interface

This UI is coded in QML (Qt Modelling Language), JavaScript and using Python.

## Prerequesities

- PyQt5
- SocketIO

## Run 

To run the program open CMD and type ``python main.py`` and when asked to input password type in ``vany``.
